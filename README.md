# 🏃 AI Running Coach — ИИ-тренер по бегу

Структурированная система для онбординга, планирования и анализа тренировок бегуна с помощью ИИ.

## Онбординг: что это и зачем

Перед тем как ИИ-тренер построит план — нужно собрать данные об атлете.
Без них план будет «среднестатистическим», а не твоим.

### 3 сессии, ~60 минут

| Сессия | Шаги | Что делаем | Время |
|---|---|---|---|
| **Сессия 1** | 1-4 | Стиль тренера → опросник → цель → методология | ~30 мин |
| **Сессия 2** | 5-8 | Локации → обувь → пульс → ОФП | ~15 мин |
| **Сессия 3** | 9-10 | Garmin → генерация плана | ~15 мин |

**Если пропустить блоки:**

| Пропущенный блок | Последствия |
|---|---|
| Общие данные | Темпы с погрешностью ±15-20% |
| Рекорды (VDOT) | Нет привязки к твоему уровню |
| Травмы | Риск обострения в первые 2-3 недели |
| Быт (график) | До 30% пропусков тренировок |
| Питание | Незначительно, может быть некомфортно на предгоночной |

## Как это работает

1. Пользователь отвечает на вопросы опросника (блок за блоком, с пояснениями)
2. ИИ заполняет `Профиль атлета.md` и `Цель и достижимость.md`
3. Проверяется достижимость цели через VDOT-таблицы Дэниелса
4. Генерируется план на 14-16 недель с учётом травм, быта и локаций
5. Если есть Garmin — настраивается API для автоматического анализа тренировок
6. В рабочем режиме — `/garmin-sync` даёт полный разбор метрик и сравнение с планом

### Методология

- **Джек Дэниелс (VDOT)** — тренировочные темпы от текущего уровня
- **80/20** — 80% лёгкого бега, 20% интенсивного
- **Норвежский метод** — короткие пороговые отрезки вместо длинных ПАНО-повторов (безопаснее для суставов)
- **Горки** — альтернатива интервалам при травмах колена

## Установка OpenCode

OpenCode — open-source ИИ-агент для терминала. **Регистрация не нужна**, встроенные бесплатные модели доступны сразу.

### Установка

```bash
# Через curl (macOS / Linux / WSL)
curl -fsSL https://opencode.ai/install | bash

# Или через npm (Windows / macOS / Linux)
npm i -g opencode-ai

# Или через brew (macOS)
brew install anomalyco/tap/opencode
```

### Бесплатные встроенные модели

OpenCode поставляется с двумя бесплатными моделями, доступными без подписки:

| Модель | Описание |
|---|---|
| **DeepSeek V4 Flash Free** | Быстрая, лёгкая, для повседневных задач |
| **Mimo V4 Free** | Умеренная, баланс скорости и качества |

Дополнительно можно подключить Claude, GPT, Gemini и другие модели через провайдеров.

### Запуск

```bash
# В папке проекта
opencode

# OpenCode прочитает 00-Context.md, затем _state.md,
# и продолжит онбординг с того места, где остановились
```

Работает в терминале, десктоп-приложении (бета) и расширениях для VS Code, Cursor, Zed.
