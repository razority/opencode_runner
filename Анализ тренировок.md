# Анализ тренировок

> Пост-онбординговый протокол.
> Основной режим работы: ежедневный цикл мониторинг → анализ → коррекция → советы.
> Выполняется при каждом синке Garmin или ручном вводе данных.

---

## Ежедневный цикл работы

```
Garmin sync / ручной ввод
        │
        ▼
┌───────────────────┐
│ 1. Собрать метрики │ ← RHR, HRV, сон, стресс, BB
└────────┬──────────┘
         ▼
┌───────────────────┐
│ 2. Оценить         │ ← Утренний фильтр (зелёный/жёлтый/красный)
│    восстановление   │
└────────┬──────────┘
         ▼
┌───────────────────┐
│ 3. Сверить с       │ ← Была ли тренировка? Качество?
│    планом          │
└────────┬──────────┘
         ▼
┌───────────────────┐
│ 4. Обновить тренды  │ ← Скользящие средние RHR, HRV, нагрузка
└────────┬──────────┘
         ▼
┌───────────────────┐
│ 5. Скорректировать  │ ← Заменить тип тренировки, снизить темп,
│    план на сегодня  │    перенести, дать отдых
└────────┬──────────┘
         ▼
┌───────────────────┐
│ 6. Дать совет      │ ← Рекомендации по тренировке,
│    атлету          │    восстановлению, питанию, сну
└───────────────────┘
```

## 1. Метрики восстановления (из Garmin)

| Метрика | Норма | Жёлтый | Красный | Действие |
|---------|-------|--------|---------|----------|
| **RHR** | ≤52 или норма для атлета | +3-5 уд/мин | +6+ уд/мин | Жёлтый → убрать интенсивность. Красный → только Z1 или отдых |
| **HRV** | ≥55 (Balanced) | 40-54 (Unbalanced) | <40 (Low) | Unbalanced → без T/Low → без T + J только |
| **Сон (общий)** | ≥7ч | 6-7ч | <6ч | <6ч → запрет на скоростные |
| **Глубокий сон** | ≥1.5ч | 1-1.5ч | <1ч | <45 мин → запрет на скоростные |
| **Body Battery (утро)** | ≥70 | 50-69 | <50 | <50 → день отдыха или лёгкий J |
| **Стресс** | <30 | 30-50 | >50 | >50 → снизить объём, убрать T |

Если нет Garmin — оценка по RPE и самочувствию атлета (шкала 1-10).

## 2. Тренды (скользящие средние)

После каждой недели обновлять:

| Параметр | Расчёт | Сигнал |
|----------|--------|--------|
| **RHR-тренд** | Средняя за 7 дней vs средняя за предыдущие 7 | Рост 3+ дня подряд → недовосстановление |
| **HRV-тренд** | Средняя за 7 дней vs базовая линия (30 дней) | Падение >10% от базовой линии → риск перетрена |
| **CTL** | Хроническая нагрузка (42 дня) | Резкий рост >10% за неделю → риск травмы |
| **ATL** | Острая нагрузка (7 дней) | ATL > CTL × 1.3 → перегрузка |
| **TSB** | CTL - ATL | TSB < -30 → глубокая усталость, разгрузка |
| **Выполнение плана** | % выполненных тренировок за неделю | <80% → пересмотреть план по загрузке |

### Правила проактивной коррекции по трендам

- RHR растёт 3+ дня подряд → снизить объём на 20%, убрать T до стабилизации
- HRV упал >10% от базовой линии → разгрузочный день
- TSB < -30 → автоматическая разгрузочная неделя, уведомить атлета
- ATL/CTL > 1.3 → заменить следующую T-тренировку на E
- Атлет пропустил 2+ тренировки подряд с болью → Vetо

## 3. Адаптация плана на лету

Коррекция применяется **до** тренировки, а не после:

| Утренний вердикт | Действие с сегодняшней тренировкой |
|-----------------|-----------------------------------|
| 🟢 Зелёный | План без изменений |
| 🟡 Жёлтый (RHR/HRV) | T → E. E → J. J → отдых. |
| 🟡 Жёлтый (сон) | Оставить E, убрать T и горки |
| 🔴 Красный | Только J или полный отдых |

Если запланирован T, а вердикт жёлтый — заменить на E или горки (меньше времени, ниже ударная).
Если запланирована длительная, а вердикт жёлтый — сократить на 30-50%, снизить темп.

## 4. Формат ответа атлету

### Блок 1. Статус восстановления

```
=== СТАТУС ВОССТАНОВЛЕНИЯ ===
Пн DD.MM — Отдых / Тренировка
  RHR: ✅/🟡/🔴 | HRV: Balanced/Unbalanced/Low | WakeBB: число
  Сон: Xч (глуб: Xч) | Стресс: число
  ВЕРДИКТ: ЗЕЛЁНЫЙ / ЖЁЛТЫЙ (без скорости) / КРАСНЫЙ (только J)
  ТРЕНДЫ: RHR стабилен/растёт | HRV стабилен/падает | TSB: число
```

### Блок 2. Разбор тренировки (если была)

```
День DD.MM — Тип тренировки
  ⏱ Xкм @ X:XX/км | Каденс XXX | HR XXX/XXX | Z2 XX%
  📊 Контакт XXXмс | Верт X.Xсм (X.X%)
  ✅ / ⚠️ / ❌ — анализ
```

### Блок 3. Совет на сегодня/завтра

```
🎯 СОВЕТ:
  - Сегодня: [тип тренировки или отдых] с [коррекция]
  - Фокус: [что отслеживать / на что обратить внимание]
  - Восстановление: [совет по сну, питанию, МФР]
```

## 5. Поля анализа тренировки

| Поле | Критерий | Оценка |
|------|----------|--------|
| HR vs зона | Средний пульс в целевой зоне | ✅ / ⚠️ / ❌ |
| Каденс | >= целевого — норма, ниже — флаг | ✅ / ⚠️ |
| Вертикальные колебания | <8.5 см — норма | ✅ / ⚠️ |
| Время контакта | ~XXXms (±10) — норма | ✅ / ⚠️ |
| TE | 2.0-3.0 (E) / 3.0-4.0 (T) | Соответствие |

## 6. Дополнительные триггеры

- Каденс устойчиво ниже нормы на равнине → добавить СБУ
- Вертикальные колебания >8.5 см → ухудшение экономичности, проверить постановку
- Время контакта >245 мс → потеря упругости, нужна плио
- TE >4.0 на аэробной → перегрузка, пересмотреть план
- Боль в суставах/связках >3/10 → Vetо, замена на ОФП

## 7. Адаптация плана (по результатам недели)

| Результат недели | Действие |
|-----------------|----------|
| Все тренировки в зелёной зоне | Следующая неделя — штатное повышение нагрузки |
| 1-2 жёлтых флага | Та же нагрузка, без прогрессии |
| Красный флаг или травма | Разгрузочная неделя вне графика |
| <80% выполнения | Пересмотреть план: снизить частоту или объём |

## 8. Погодная поправка

Перед анализом сверить погоду на момент тренировки (Open-Meteo).
При экстремальных температурах (<0°C или >30°C) — применять сезонную адаптацию из `Методология.md` (раздел 5). Темп и пульс в таких условиях не сравнивать с плановыми зонами без коррекции.

- **Garmin Epix Pro Gen 2** + `garminconnect`
- **Polar H10** (нагрудный пульсометр)
- **Ручные данные** (если нет Garmin)
