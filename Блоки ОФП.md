# Блоки ОФП

> Заполняется на Шаге 8 онбординга.
> Блоки подбираются под слабые места атлета и компенсацию беговых нагрузок.

---

## Оценка необходимости СБУ

| Критерий | Если да → добавить СБУ |
|---|---|
| Каденс <170 на E | Да |
| Вертикальные колебания >9 см | Да |
| Время контакта >250 мс | Да |
| Частые травмы голеностопа / ахилла | Да |
| Нет опыта ОФП | Умеренно |

## Структура блоков (принцип)

| Блок | Назначение | Когда |
|---|---|---|
| **Блок А — Стабилизация** | Колени, таз, стопа | После каждой беговой тренировки (среды и пятницы) |
| **Блок Б — Задняя цепь** | Ягодицы, задняя поверхность бедра | Со 2-го месяца базы, 2 раза в неделю |
| **Блок В — Плио + Ахилл** | Упругость стопы, взрывная сила | С начала спецпериода |
| **Блок Г — Кор** | Пресс, поясница, устойчивость | В дни отдыха / ОФП |
| **Растяжка** | TFL, квадрицепс, икры, сгибатели бедра | После каждой беговой |
| **МФР** | TFL, ягодицы, квадрицепс, икры | Дни отдыха, вечером |
