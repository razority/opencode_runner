# Блоки ОФП

> Заполняется на Шаге 8 онбординга.
> Блоки подобраны под профиль: ITBS, опыт ОФП есть, СБУ — с нуля.

---

## Оценка необходимости СБУ

| Критерий | Решение |
|---|---|
| Каденс | Пока не известен — замерить на первых беговых |
| ITBS | Нужна стабилизация колена (Блок А) |

## Структура блоков

| Блок | Назначение | Когда |
|---|---|---|
| **Блок А — Стабилизация колена** | VMO, таз, стопа | После бега в Ср + Пт |
| **Блок Б — Задняя цепь** | Ягодицы, задняя поверхность | С 3-й недели, 2 раза в нед |
| **Блок В — СБУ (введение)** | Семенящий, бедро, захлёст | После разминки перед качественной |
| **Блок Г — Кор** | Пресс, поясница | Пн / после бега, 2-3 раза в нед |
| **Растяжка** | TFL, квадрицепс, икры, сгибатели бедра | После каждой беговой (уже делаешь ✅) |
| **МФР** | Перкуссионный массажёр | Дни отдыха, вечером |

---

## Программа на первые 4 недели (базовый период)

### Блок А — Стабилизация колена (Ср + Пт после бега)

| Упражнение | Дозировка | Примечание |
|---|---|---|
| «Стульчик» у стены | 3 x 45-60" | Мяч между коленями для VMO |
| Подъём прямой ноги лёжа | 3 x 15-20 | Носок на себя, задержка вверху |
| «Книжка» (Clamshell) | 3 x 20-30 на сторону | Средняя ягодичная — разгрузка колена |
| Баланс на одной ноге | 3 x 30-45" на ногу | Устойчивость голеностопа |

### Блок Б — Задняя цепь (со 2-го месяца)

| Упражнение | Дозировка | Примечание |
|---|---|---|
| Ягодичный мостик на одной ноге | 3 x 15 | Опора на пятку |
| Румынская тяга на одной ноге | 3 x 12 | Спина прямая, таз назад |

### Блок В — СБУ (перед качественными тренировками, после разминки)

| Упражнение | Дозировка | Примечание |
|---|---|---|
| Семенящий бег | 2 x 30м | Часто, низко |
| Бег с высоким подниманием бедра | 2 x 30м | Средний темп |
| Захлёст голени | 2 x 30м | Пятки к ягодицам |

### Блок Г — Кор (Пн, Пт)

| Упражнение | Дозировка | Примечание |
|---|---|---|
| Планка на локтях | 3 x 45-60" | Поясница не провисает |
| Боковая планка | 3 x 30-45" на сторону | |
| «Птица-собака» | 3 x 12-15 на диагональ | |

### Растяжка (после бега — обязательно, уже делаешь)

| Упражнение | Дозировка |
|---|---|
| Икры у стены | 45-60" на ногу |
| Квадрицепс стоя | 30-45" на ногу |
| Сгибатели бедра (TFL) | 45" на ногу |

### Блок Д — Плио (при необходимости, по данным анализа)
| Упражнение | Дозировка | Примечание |
|---|---|---|
| Прыжки на месте с подъёмом колен | 3 x 10-15 | Мягкое приземление, колено стабильно |
| Выпрыгивания из полуприседа | 3 x 8-10 | Акцент на взрывную силу |
| Скакалка | 3 x 30-60" | Развитие упругости стопы |

### МФР (дни отдыха, перкуссионный массажёр)

| Зона | Время | Примечание |
|---|---|---|
| TFL (сбоку-спереди таза) | 2-3' | Мягко, не давить на кость |
| Квадрицепс (внешняя часть) | 2' | |
| Ягодицы | 2' на сторону | |
| Икры | 2' на ногу | |
