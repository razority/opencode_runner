# Глоссарий

> Термины, которые использует тренер в общении.
> Если термин непонятен — атлет может открыть этот файл.

| Термин | Расшифровка |
|---|---|
| **ПАНО** | Порог анаэробного обмена. Пульс, при котором организм переключается с аэробного на анаэробное энергообеспечение. Ключевая метрика для построения зон. |
| **VDOT** | Показатель эффективности бега (по Дэниелсу). Один VDOT = один уровень подготовки. По VDOT рассчитываются тренировочные темпы для всех зон. |
| **T (Threshold)** | Пороговый темп. Бег на уровне ПАНО (Z4). Основной инструмент норвежского метода. |
| **E (Easy)** | Аэробный бег (Z2 верх). «Разговорный» темп. |
| **J (Jog)** | Восстановительный бег (Z2 низ / Z1). Очень медленно, пульс минимальный. |
| **I (Interval)** | Интервалы. Жёсткие отрезки 3-5 мин с полным восстановлением. В текущем протоколе заменены на норвежский метод (T-отрезки) и горки. |
| **R (Repetition)** | Повторы. Короткие взрывные отрезки (<2 мин). В текущем протоколе не используются — только после базовой подготовки, по указанию тренера. |
| **Горки** | Бег в подъём. Замена интервалам для травмобезопасного развития интенсивности. Спуск — шагом. |
| **Норвежский метод** | Пороговая работа короткими сериями с минимальным отдыхом. Вместо классических ПАНО-повторов. |
| **80/20** | Правило: 80% лёгкого бега, 20% интенсивного. |
| **CTL (Chronic Training Load)** | Хроническая тренировочная нагрузка. Показатель «натренированности» за 42 дня. |
| **ATL (Acute Training Load)** | Острая нагрузка за 7 дней. Показатель усталости. |
| **TSB (Training Stress Balance)** | CTL - ATL. Показывает, в минусе вы (усталость) или в плюсе (суперкомпенсация). |
| **RHR** | Resting Heart Rate. Пульс покоя. Главный маркер восстановления. |
| **HRV** | Heart Rate Variability. Вариабельность сердечного ритма. Показывает готовность ЦНС к нагрузке. |
| **TE (Training Effect)** | Тренировочный эффект (1.0-5.0). Показывает, как тренировка повлияла на организм. |
| **СБУ** | Специальные беговые упражнения (семенящий, с высоким подниманием бедра, захлёст голени и т.д.). |
| **МФР** | Миофасциальный релиз. Самомассаж с роллом / мячом. |
| **ITBS** | Iliotibial Band Syndrome. Синдром трения IT-тракта. Частая травма бегунов. |
| **VMO** | Vastus Medialis Obliquus. Внутренняя головка квадрицепса, стабилизатор колена. |
| **TFL** | Tensor Fasciae Latae. Напрягатель широкой фасции. Часто гипертонирован у бегунов. |
