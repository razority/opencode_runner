# Опросник атлета

> Заполняется на Шаге 2 онбординга.
> Если блок пропущен — качество плана снижается. Все предупреждения указаны ниже.

---

## Блок 1. Общие данные

**Зачем:** Возраст, вес, пульс покоя — база для расчёта зон ЧСС по формуле 180 - возраст. Образ жизни определяет время восстановления между тренировками.

**Глоссарий:** `RHR` (пульс покоя) — главный маркер восстановления. `ПАНО` — пульс анаэробного порога, от него строятся все зоны.

**Если пропустить:** Без этого блока план строится «вслепую» — зоны ЧСС будут рассчитаны от среднестатистических значений, а не от твоих. Погрешность тренировочных темпов: **±15-20%**.

_Готов предоставить эти данные? (для точности плана рекомендуется)_

| Вопрос | Для чего |
|---|---|
| Возраст | Расчёт аэробного коридора |
| Вес | Энергозатраты, темп на длительных |
| Пол | Физиологические нормы |
| Род деятельности | Влияет на утомление ЦНС |
| Режим сна и часы | Корректировка нагрузки |

## Блок 2. Беговой стаж и рекорды

**Зачем:** Текущие результаты — единственный объективный способ рассчитать VDOT (твой уровень подготовки). VDOT определяет тренировочные темпы для каждой зоны.

**Глоссарий:** `VDOT` — число, которое связывает все твои рекорды в одну шкалу. Каждому VDOT соответствует свой темп для лёгкого (E), порогового (T), интервального (I) бега.

**Если пропустить:** Тренировочные темпы будут подобраны «на глаз», без привязки к твоему реальному уровню. Риск: слишком лёгкий бег (нет прогресса) или слишком быстрый (травма / перетрен). Погрешность: **±20%**.

_Готов предоставить рекорды? (хотя бы на одну дистанцию)_

| Вопрос | Для чего |
|---|---|
| Сколько лет бегаешь | Скорость адаптации к нагрузкам |
| Тренировок в неделю | Отправная точка объёма |
| Недельный объём (км) | База для прогрессии |
| Рекорд на 5 км | VDOT-калькуляция |
| Рекорд на 10 км | VDOT-калькуляция |
| Рекорд на полумарафон | VDOT-калькуляция |

## Блок 3. Текущая форма

**Зачем:** Понимаем, с какой точки стартуем — с пика, с базы или с нуля. Это определяет темп вводной недели.

**Глоссарий:** `J` (Jog) — восстановительный бег, самый медленный. `E` (Easy) — аэробный, разговорный темп.

**Если пропустить:** План может начаться с завышенного объёма (срыв в первые 2 недели) или с заниженного (потеря времени). Погрешность: **±1-2 недели в начале**.

_Готов оценить текущую форму?_

| Вопрос | Для чего |
|---|---|
| Самооценка формы (пик / база / с нуля) | Точка входа в план |
| Последний старт на результат | Текущий пик формы |
| Темп лёгкого бега | Якорь для Z2 |
| Бегал ли на прошлой неделе | Недельный контекст |

## Блок 4. Травмы и ограничения

**Зачем:** Самый важный блок. Из него вытекают все veto-правила, запреты и лимиты. Без него план может спровоцировать обострение.

**Глоссарий:** `ITBS` — синдром трения IT-тракта, частая травма бегунов. `VMO` — внутренняя головка квадрицепса, стабилизатор колена. `TFL` — напрягатель широкой фасции, частая зона гипертонуса.

**Если пропустить:** План будет сгенерирован как для здорового атлета. При скрытой травме — обострение в первые 2-3 недели. Рекомендуется **не пропускать**.

_Готов рассказать о травмах и ограничениях?_

| Вопрос | Для чего |
|---|---|
| Хронические травмы | Veto-правила, запрет типов работ |
| Операции | Ограничения по ОФП |
| Бег через боль | Коррекция поведенческого паттерна |
| Разминка / растяжка | Включение в план |

## Блок 5. Быт и режим

**Зачем:** График — главный убийца плана. Легче встроить тренировки в реальную жизнь, чем потом догонять пропуски.

**Глоссарий:** `ATL` — острая нагрузка (усталость недели). `CTL` — хроническая нагрузка (натренированность). Если быт мешает спать/есть — ATL растёт быстрее CTL.

**Если пропустить:** План может требовать бега в дни, когда это объективно невозможно. Результат: срыв графика и чувство вины. Погрешность: **до 30% пропусков**.

_Готов согласовать график?_

| Вопрос | Для чего |
|---|---|
| Удобное время бега | Привязка типов тренировок |
| Сколько дней готов бегать | Количество тренировок в неделе |
| Заблокированные дни | Никаких планов на эти дни |
| Алкоголь | Корректировка нагрузки на следующий день |

## Блок 6. Питание и добавки

**Зачем:** Для предсоревновательной загрузки, управления весом и коррекции энергодефицита на длительных.

**Глоссарий:** `Изотоники` — напитки для восполнения электролитов на длительных (>60 мин). `Бета-аланин` — добавка для буферизации лактата.

**Если пропустить:** План всё равно будет рабочим, но предгоночная неделя (углеводная загрузка) не будет адаптирована под твои привычки. Погрешность: **незначительная, но на предгоночной неделе может быть некомфортно**.

_Готов рассказать о питании?_

| Вопрос | Для чего |
|---|---|
| Диета / ограничения | Адаптация предгоночной загрузки |
| Добавки | Корректировка циклирования |
| Изотоники на длительных | Стратегия гидратации |
| Контроль веса | Прогноз темпа |
