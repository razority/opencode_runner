# Опросник атлета

> Каждый блок предварять пояснением «Зачем это нужно».
> После заполнения — перенести данные в `Профиль атлета.md`.

---

## Блок 1. Общие данные

**Зачем:** Возраст, вес, образ жизни влияют на восстановление, риски травм и выбор зон ЧСС.

| Вопрос | Пример ответа |
|---|---|
| Возраст | 32 |
| Рост | 175 см |
| Вес | 68 кг |
| Пол | М / Ж |
| Род деятельности | Офисная работа / физический труд / IT-удалёнка |
| Режим сна | Во сколько ложится / встаёт |
| Средняя продолжительность сна | 7 ч |
| Есть ли хронический стресс | Да / Нет / Умеренный |

## Блок 2. Беговой стаж и рекорды

**Зачем:** Определяем текущий VDOT, зону ближайшего роста, скорость адаптации к нагрузкам.

| Вопрос | Пример ответа |
|---|---|
| Сколько лет регулярно бегаешь? | 5 лет |
| Сколько тренировок в неделю? | 4-5 |
| Текущий недельный объём (км/ч) | 50 км / 4.5 ч |
| Как давно на этом объёме? | 3 месяца |
| Личные рекорды на 1 км | 3:40 |
| Личные рекорды на 5 км | 19:30 |
| Личные рекорды на 10 км | 41:00 |
| Личные рекорды на полумарафон | 1:28:00 |
| Личные рекорды на марафон | 3:10:00 |
| Когда и при каких условиях установлены последние рекорды? (погода, профиль, самочувствие) | Осень 2025, +10°C, равнина |

## Блок 3. Текущая форма

**Зачем:** Понимаем, с какой точки стартуем.

| Вопрос | Пример ответа |
|---|---|
| Есть ли ощущение текущей формы? | Пик / База / «С нуля» / Восстановление после перерыва |
| Когда был последний забег на результат? | 2 месяца назад |
| Бегал(а) ли на прошлой неделе? | Да / Нет |
| Как оцениваешь текущий темп на лёгком беге? | 5:30-5:45 мин/км |

## Блок 4. Травмы и ограничения

**Зачем:** Это самый важный блок. Из него вытекают veto-правила и ограничения интенсивности.

| Вопрос | Пример ответа |
|---|---|
| Есть ли хронические травмы? | ITBS, Ахилл, Подошвенный фасциит, Боль в спине |
| Были ли операции? | Нет |
| Бегаешь ли сейчас с болью? | Нет / Иногда / Да |
| Есть ли проблемы с сердцем / давлением? | Нет |
| Есть ли ограничения по ОФП (колени, спина, плечи)? | Не могу приседать глубоко |
| Делаешь ли разминку перед бегом? | Да / Нет / Иногда |
| Делаешь ли растяжку после? | Да / Нет / Иногда |

## Блок 5. Быт и режим

**Зачем:** Работа, семья, быт — ключевые факторы срыва плана. Их лучше учесть сразу.

| Вопрос | Пример ответа |
|---|---|
| Удобное время для тренировок | Утро / День / Вечер |
| Сколько дней в неделю готов(а) бегать | 4-5 |
| Есть ли дни, когда бегать точно не получится? | Среда (поздняя работа) |
| Алкоголь? Как часто? | 1 раз в неделю / не пью |
| Курение? | Нет |
| Есть ли дети / другие обязательства, влияющие на график? | Да / Нет |

## Блок 6. Питание и добавки

**Зачем:** Для коррекции плана в предсоревновательный период и управления весом.

| Вопрос | Пример ответа |
|---|---|
| Есть ли диета / ограничения? | Веганство / Без лактозы / Нет |
| Принимаешь ли добавки? (BCAA, креатин, цитруллин, бета-аланин) | Цитруллин малат |
| Пьёшь ли изотоники на длительных? | Да / Нет |
| Контролируешь ли вес? | Да / Нет |
