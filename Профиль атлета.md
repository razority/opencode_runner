# Профиль атлета

> Заполняется из `Опросник атлета.md` после Шага 2 онбординга.
> Все ограничения из этого файла переходят в veto-правила тренера.

---

## Общие данные

| Параметр | Значение |
|---|---|---|
| Возраст | 38 |
| Рост | 171-180 см |
| Вес | 76 кг |
| Пол | М |
| Работа | IT / удалёнка (сидячая) |
| Режим сна | 00:00-02:00 / 08:00-10:00 |
| Средний сон | 8+ ч |
| Уровень стресса | Умеренный |

## Беговой стаж

| Параметр | Значение |
|---|---|
| Лет бега | 5-10 |
| Тренировок в неделю | 4-5 |
| Недельный объём (км) | 50-60 |
| Недельный объём (часы) | ~4.5-5 ч |
| Стабильность объёма | 4 года |

## Личные рекорды

| Дистанция | Время | Темп | Условия |
|---|---|---|---|
| 5 км | 18:20 | 3:40 | В разное время года |
| 10 км | 36:45 | 3:40 | В разное время года |
| Полумарафон | 1:21:00 | 3:50 | В разное время года |
| Марафон | 3:00-3:14 | ~4:15-4:35 | — |

## Текущая форма

| Параметр | Значение |
|---|---|
| Самооценка формы | База (набор объёмов) |
| Последний старт | Месяц назад |
| Темп лёгкого бега (J) | 5:00-5:15 |
| Бег на прошлой неделе | Да, по плану |

## Травмы и ограничения

| Травма | Статус | Ограничения |
|---|---|---|
| ITBS (колено, внешняя сторона) | Активно (эпизодически) | Нет спусков бегом, лимит объёма, без R/I работ |
| Разминка | Не делает | Требуется внедрить |
| Иногда бегает через слабую боль | — | Требуется контроль |

## Быт

| Параметр | Значение |
|---|---|
| Удобное время тренировок | Утро или обед (будни), любое (выходные) |
| Доступные дни | 5 дней в неделю |
| Заблокированные дни | Четверг |
| Алкоголь | 1 раз в неделю |
| Другие обязательства | Нет |

## Питание

| Параметр | Значение |
|---|---|
| Диета | Без ограничений |
| Добавки | Нет |
| Изотоники | Нет (только вода) |
| Контроль веса | Иногда |
