# Пульсовые зоны

> Заполняется на Шаге 7 онбординга.
> Если ПАНО неизвестен — рекомендовать тест Джо Фрила.

---

## Зоны (от ПАНО)

| Зона | % от ПАНО | Пульс | Назначение |
|---|---|---|---|
| Z1 | 62-79% | | Восстановление, разминка, заминка |
| Z2 | 80-89% | | Аэробный бег (J / E) |
| Z3 | 90-94% | | Развивающий кросс, темп |
| Z4 | 95-102% | | Пороговый коридор (T) |
| Z5 | >103% | | Анаэробный резерв |

## 30-минутный тест Джо Фрила (если ПАНО неизвестен)

### Протокол

1. Разминка 15' лёгким бегом
2. 30 минут бега на максимуме («как на 10 км»)
3. **Средний пульс за последние 20 минут** = ПАНО

### Требования
- Ровный рельеф (стадион или набережная)
- Хорошее самочувствие, RHR в норме
- Не делать на жаре (>25°C)
- За 2 дня до теста — без интенсивных тренировок

## Если тест провести нельзя

**Формула 180 - возраст (+ поправки):**
- +5, если тренируешься >2 лет без перерывов
- +0, если тренируешься 1-2 года
- -5, если есть травмы или болезни

→ Полученное число = верхняя граница Z2 (аэробный порог).
→ ПАНО примерно на 15-20 уд выше аэробного порога (уточняется на первых T-тренировках).
