# Пульсовые зоны

> Заполняется на Шаге 7 онбординга.
> После расчёта зон — перенести ПАНО, Z2 и T-темп в `Профиль атлета.md` (вычисляемые параметры).
> Если ПАНО неизвестен — рекомендовать 30-минутный тест Джо Фрила.

---

## Зоны (от ПАНО)

| Зона | % от ПАНО | Пульс | Назначение |
|---|---|---|---|
| Z1 | 62-79% | | Восстановление, разминка |
| Z2 | 80-89% | | Аэробный (J / E) — J: 80-84%, E: 85-89% |
| Z3 | 90-94% | | Развивающий |
| Z4 | 95-102% | | Пороговый (T) |
| Z5 | >103% | | Анаэробный резерв |

## 30-минутный тест Джо Фрила

1. Разминка 15' лёгким бегом
2. 30 минут бега на максимуме («как на 10 км»)
3. **Средний пульс за последние 20 минут** = ПАНО

**Требования:** ровный рельеф, хорошее самочувствие, не на жаре, за 2 дня без интенсивности.

## Приоритет определения ПАНО

Использовать первый доступный метод (от лучшего к худшему):

| Приоритет | Метод | Точность |
|---|---|---|
| 1 | Лабораторный тест (газоанализ) | Эталон |
| 2 | 30-минутный тест Джо Фрила (выше) | Высокая |
| 3 | Известное значение с прошлых тренировок / гонок | Средняя |
| 4 | Формула **180 - возраст** (+ поправки для стажа) | Низкая, но допустима |

Рассчитанный ПАНО сразу перенести в `Профиль атлета.md` (вычисляемые параметры).
